# TechStore 🛍️

Modern e-commerce app built with Next.js 15, TypeScript, Tailwind CSS, and Zustand.

## Features
- Product catalog with search & filter
- Product detail pages
- Shopping cart with quantity controls
- Checkout flow
- Responsive design

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- Zustand (state management)
- Lucide React (icons)
